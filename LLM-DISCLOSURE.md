# LLM disclosure

This project may include contributions written with assistance from large language models (LLMs) or similar tools.

## Expectations for contributors

- **Accuracy:** You are responsible for verifying that AI-suggested changes build, run, and behave correctly in this Docker Compose context.
- **Security:** Do not paste real secrets, production credentials, or customer data into AI tools.
- **Licensing:** Ensure you have the right to submit any suggested code or text under this project’s license.
- **Transparency:** When opening a pull request, it is helpful to note significant AI assistance in the PR description so reviewers can focus on validation.

## For maintainers

AI-generated patches should meet the same review bar as human-written patches: correctness, minimal scope, and alignment with [README.md](README.md) and [AGENTS.md](AGENTS.md).

This file is not legal advice; it documents project norms only.
